一般这种迁移，要注意旧仓库的提交历史等信息也要同步到新的仓库。

先使用如下命令克隆老的：

```sh
git clone --bare git@gitlab.test1.com:f2e/test.git
```

新仓库创建好，使用如下命令推上去：

```sh
git push --mirror git@gitlab.test2.com:f2e/test.git
```

这样旧仓库当中的提交历史，以及各个分支标签就会同步过来了。
